# Axl's Interactive Portfolio

Welcome to my interactive portfolio. This Minecraft Bedrock Edition add-on showcases a small collection of custom items, blocks, and mechanics created by me.
The main goal of this addon is to showcase **_refined, robust, re-usable and general algorithms_** that I've integrated into my personal library.

The primary focus was to obtain the **_highest quality_** possible making the features as close as possible to vanilla systems (including creative/survival interactions) and making them **_resilient_** to most edge cases.

To learn more about what project I've worked on, refer to my [Web Portfolio](https://quantumnek0.github.io/Quantum-Cube-Studios/).

## Features

### Crop System

- **Diamond Flower**: Re-creation of the vanilla crop system with ambient particles and harvest mechanics.
- **Diamond Flower Seeds**: Used to cultivate Diamond Flowers.

---

### Potion System

- **Haste Potion**: Basic effect system for consuming items.

---

### Ore Behavior

- **Diamond Dust Ore**: Re-creation of the vanilla ore behavior with enchantment support.
- **Diamond Dust**: Obtained from Diamond Dust Ore.

---

### Tree Capitator Algorithm

- **Gravity Axe**: Mines every log connected to a given tree (Tree Capitator Algorithm). Supports enchantments.

---

### Dynamic Lighting System

- **Diamond Torch**: Dynamic lighting system that places and removes light blocks following the player. Supports off-hand use.

---

### Custom TNT

- **Lightning TNT**: Custom TNT that spawns lightning bolts around the area.

---

### TNT Pickaxe

- **TNT Pickaxe**: Creates a 3x3x3 explosion after a block is mined.

---

### Structures

- **The Backrooms**: Generates a jigsaw structure with a custom theme and music.

---

## Installation

1.  Download the [.mcaddon](Portfolio.mcaddon) file
2.  Double-click the downloaded file to import into Minecraft.
3.  Apply the Resource Pack and Behavior Pack to any world.
